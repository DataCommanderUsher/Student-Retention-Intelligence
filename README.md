# Student-Retention-Intelligence
A data-driven analysis of student demographics and academic performance metrics to identify key indicators of attrition. This project utilizes historical data to isolate risk factors and proposes a targeted intervention strategy to mitigate dropout rates and improve institutional retention.

## 1. A clear statement of the business task
Analyze historical student performance and demographic data to identify the primary correlation factors contributing to student dropout and failure. These insights will be used to design a data-backed 'Early Warning System' strategy, enabling administration to intervene with at-risk students before they churn, thereby increasing overall retention rates.
